"id" is lerner id.
"avg" is Average of total assigment scores divided to total possible points allowed.
lis of "number of assignment id" is an average of each assigment score divided to a possible point allowed.

Check the assignment group if it has valid course information.
Check the possible point in list of assigments if it is 0 point allowed to avoid zero division.
Use "try-catch" block feature to handle any exceptions to avoid fatal error causing system termination.
An assignment submitted early meaning before the due date, will not include in the report.
An assignment submitted late to due date will be deducted 10% from the assignment score.
